# Bot Identicon
A React component that deterministically generates robot icons based on the given input. See example code below:
```jsx
<BotIdenticon 
  identifier="hej"
  size={500}
/>
```

## Props
* `identifier` - the default is an empty string
* `size`       - the size of the icon, default is 200px
